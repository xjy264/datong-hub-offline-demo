# 删除界面坐标展示设计

## 目标

删除系统界面中面向用户展示的车站坐标文字，同时保留地图定位、拖拽、缩放、搜索聚焦和区间连线所需的内部坐标数据。

## 设计

- “编辑车站区间”的车站 A、车站 B 下拉选项只显示车站名称，不再附带 `(x, y)`。
- 当前车站区间右侧栏保持只读，不改动现有展示和交互。
- 后端接口、数据库坐标字段、前端坐标类型和地图计算逻辑全部保留。
- 全局检查其他用户可见界面；若不存在其他坐标文字，不扩大修改范围。

## 验证

- 增加标签格式测试，确认区间车站选项只返回车站名称。
- 运行全部前端测试和构建。
- 在本地 `/map` 进入编辑布局并选中区间，确认两个下拉框没有坐标文字，地图位置和区间连线不变。
