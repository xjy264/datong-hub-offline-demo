大同示意图 Windows 部署操作手册

1. 部署前准备

使用完整离线包 datong-map-windows-offline.zip，解压到 C:\DatongMap。准备已有MySQL管理员凭据。服务器建议8GB以上内存、系统盘20GB以上可用空间。

2. 开始部署

双击 开始部署.cmd，权限确认选择“是”。向导按01到05执行，每阶段成功后输入Y继续。

绿色PASS

当前阶段通过，可以进入下一步。

黄色提醒

阅读提示并确认配置；例如内存低于推荐值或复用MySQL监听范围由甲方管理。

红色STOP

停止操作，把reports目录或诊断ZIP发送给远程技术人员。

3. MySQL选择

  • 一个MySQL 8.x：按Enter复用。
  • 多个实例：输入报告中的编号。
  • 旧MySQL、MariaDB或无兼容服务：输入0，使用项目独立MySQL。
  • 管理员密码输入时屏幕不显示字符，凭据只用于本次建库。

4. 成功标志

  • environment-report.html为PASS。
  • deployment-result.html为PASS。
  • 项目服务为Running，首页与健康检查为200。

5. 客户端

复制client目录和 客户端证书安装.cmd 到每台客户端，双击并输入服务器名称、局域网IP，随后访问 https://服务器IP:8012。

6. 异常处理

C:\DatongMap\scripts\collect-diagnostics.ps1

把生成的DatongMap-Diagnostics ZIP发送给远程技术人员。

7. 日常维护

status.ps1
start.ps1
stop.ps1
restart.ps1
backup.ps1
restore.ps1 -BackupPath "备份路径"

自动保留7个日备份和4个周备份。

8. 卸载

uninstall.ps1

默认保留数据和复用的甲方MySQL服务。完整细节请查看同目录Markdown手册。
